Acerca del sistema

Este sistema web fue creado para realizar la estimación de altura en el diseño de obra. La principal característica es que se utiliza la tecnología de un Sistema de Información Geográfica basada en la web, esto permite a los usuarios crear consultas interactivas, integrar, analizar y representar de una forma eficiente cualquier tipo de información geográfica referenciada asociada a un predio, conectando mapas con las bases de datos consiguiendo realizar los cálculos necesarios para generar la estimación esperada.

Colaboradores

Para la realización de este sistema participaron distintas entidades, una de ellas fue el centro tecnológico de hidrología ambiental de la universidad de Talca, otro colaborador es la empresa Masisa para la cual está dirigido el sistema, por último este proyecto fue guiado por el Profesor Dr. Benjamin Ingran de la universidad de Talca y desarrollado por el alumno Fernando Gómez, con la ayuda del los alumnos L Rodrigo Pozo y Mauricio Orellana.
